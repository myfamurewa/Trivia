# Apex Quizer

A simple trivia app that is built using React and the open trivia database https://opentdb.com/.
This application let's users pick the difficulty, number and topic of their questions
if the the user locks in their options without selecting difficulty, number or topic they will
be given 10 questions of various difficulties, from random topics.
